# Ventajas y Limitaciones

### Ventajas del modelo documental

* Reducción de joins
* Mayor velocidad en lecturas de entidad completa
* Evolución flexible del modelo
* Adecuado para aplicaciones web modernas

### Limitaciones

* Duplicación de datos cuando se embebe en exceso
* Relaciones complejas menos eficientes que en grafos
* Control transaccional históricamente limitado frente a SQL clásico

### Cuándo usar MongoDB

* Catálogos con atributos variables
* Aplicaciones con evolución frecuente del modelo
* APIs que consumen y devuelven JSON
* Sistemas donde cada entidad es naturalmente autocontenida
